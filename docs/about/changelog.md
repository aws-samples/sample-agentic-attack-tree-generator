# Release Notes

## Upgrading

To upgrade ThreatForest to the latest version:

**Using pipx:**
```bash
pipx upgrade threatforest
```

**Using pip:**
```bash
pip install --upgrade threatforest
```

**From source:**
```bash
cd ThreatForest
git pull
pip install --upgrade .
```
